# Changelog

## 1.0.0 - 2026-05-14

- Live monitor records matches in the background and creates a post-game review when the match ends.
- Champion, role/lane, queue type, and stats are auto-detected from League data where Riot exposes it.
- ARAM matches are ignored so review history is not polluted by for-fun games.
- Separate tabs for Mistakes, Improve, Strengths, Focus, Training, History, and What's New.
- Pro Habits tab with high-level fundamentals and role-specific pro rules for Top, Jungle, Mid, ADC, and Support.
- Afrikaans/English language selector.
- GitHub Releases update support with installer and portable builds.
